*** what is operator in python??***


***definition***

operator operand some action there we applied operator 
examples:- addition| substraction| compare| conditional value check 


***types of operator***

1. airthematic operator:- performing some airthmatical expression there we used airthematic operator 
                          examples:- +,-, *, /, %etc.
2. comparision operator :- where we compare with 2 values either its true or false applied comparision operator .
                          examples:- >,>=,<,<=,===etc.
3. assignment operator :- where we compared and assigned two values and check condition there we applied assignment operator . examples:- =,==,!=etc.
4. logical operator :- and|or
5. ternary or conditional operator :- python is not official use ternary operator but yes we used conditional operator here examples:- if true "else" " false"
6. betwise operator : examples +=,*=,%=,/=,>>,<<
